## Setup and Installation

### Clone the Repository
* `git clone git@github.com:iamronakgupta/doctic.git`
* `cd doctic`

### Run Bundle

* `bundle install` - install all gems

### Setup Postgres
* `rails db:create`
* `rails db:migrate`
* `rails db:seed`

### Start the Server:
* `bin/dev` - starts rails server at 3000 port
